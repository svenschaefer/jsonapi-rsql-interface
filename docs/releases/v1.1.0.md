# Release Notes - `v1.1.0`

Status: draft

## Release Identity

- Package: `jsonapi-rsql-interface`
- Target version: `1.1.0`
- Planned tag: `v1.1.0`

## Changes Summary

### Added

- Policy-gated wildcard semantics for `string` + `==`.
- Deterministic wildcard modes:
  - `contains` (`*value*`)
  - `starts_with` (`value*`)
  - `ends_with` (`*value`)
- Deterministic wildcard error codes:
  - `wildcard_not_allowed`
  - `wildcard_operator_not_allowed`
  - `wildcard_type_not_supported`
  - `invalid_wildcard_pattern`

### Changed

- Wildcard behavior is now explicit extension capability (policy-gated), not implicit equality behavior.
- Contract tests expanded for wildcard positive/negative behavior and canonical error mapping.
